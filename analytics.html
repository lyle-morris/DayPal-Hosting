DayPal Analytics Dashboard

QA view for config-page opens. This is local-browser analytics, not a production multi-user analytics service.

What the numbers mean

Unique user
The first time this browser opens the DayPal config page. DayPal creates a local anonymous ID and stores the first-open time.
Return user
The same local anonymous ID opens the config page again. One user can have many return opens.

Config page opens

Primary events come from config_opened. Storage keys: daypal_analytics_events, daypal_analytics_user_id, daypal_analytics_open_count.
